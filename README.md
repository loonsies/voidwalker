# voidwalker

**voidwalker** is a small helper addon for Ashita v4 designed to assist with Voidwalker NMs.

# Features

- Displays a line pointing toward the last known Voidwalker NM location
- Shows a window with:
  - Direction to the NM
  - Distance from your current position

# Bypass heal (:warning: Caution)

- Optional **Bypass Heal** feature
- This feature blocks the *healing animation packet*
- When enabled, other players can visibly see you moving while sitting
- **Use with extreme caution**
  - Not recommended for regular use
  - Enabling this feature is noticeable to other players. They can see you moving while being in the healing animation, which is not possible normally.
